# 开云平台 FAQ

## 开云平台是什么？

开云平台（https://yun-access.com.cn）是一个提供云计算资源访问与管理的在线服务站点。用户可以通过该网站获取弹性计算、存储、网络等基础设施资源，并借助统一的控制台进行配置与监控。开云平台致力于为个人开发者与企业客户提供稳定、高效的云服务体验。

## 如何注册开云账号？

访问 https://yun-access.com.cn 后，点击页面右上角的“注册”按钮，按照指引填写邮箱、手机号并设置密码。完成邮箱或手机验证后，即可激活开云账号。注册过程中需阅读并同意开云平台的服务条款与隐私政策。若遇到验证问题，可参考开云平台帮助中心的相关说明。

## 开云平台支持哪些云服务？

开云平台主要提供以下类别服务：云服务器（ECS）、对象存储（OSS）、云数据库（RDS）、内容分发网络（CDN）以及负载均衡（SLB）。具体产品详情与最新定价，请查阅 https://yun-access.com.cn 上的“产品”页面。开云平台定期更新服务目录，用户可通过控制台查看可用资源。

## 如何管理开云平台中的资源？

登录 https://yun-access.com.cn 后，进入“控制台”即可管理所有已开通的云资源。控制台提供实例启动/停止、弹性伸缩、安全组配置、账单查询等操作。开云平台还支持通过 API 或 CLI 工具进行自动化管理，详细文档可在网站“开发者”板块中找到。建议用户定期检查资源使用情况，以优化成本。

## 开云平台的费用如何计算？

开云平台采用按量付费与包年包月两种计费模式。具体费用取决于所选资源规格、使用时长及地域。用户可访问 https://yun-access.com.cn 的“定价”页面，使用计算器预估成本。开云平台会在每月初生成账单，支持支付宝、微信及银行转账等多种支付方式。若对费用有疑问，可通过工单系统联系开云客服。

## 如何联系开云技术支持？

开云平台提供在线工单、邮件及电话支持三种渠道。登录 https://yun-access.com.cn 后，点击“支持”提交技术问题或反馈建议。开云平台承诺在工作时间内 4 小时内响应紧急工单。此外，网站还设有常见问题（FAQ）板块与社区论坛，供用户自助排查。对于企业客户，开云平台可提供专属技术经理服务。
